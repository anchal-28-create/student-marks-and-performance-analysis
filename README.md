# student-marks-and-performance-analysis
1. Student Marks & Performance Analysis System

An interactive Python-based data analytics project that analyzes student academic performance using subject-wise and student-wise analysis, generates visual insights, assigns grades, and produces a comprehensive Excel report.

2. Project Objectives
Analyze student marks from a CSV dataset
Perform subject-wise and student-wise performance analysis
Identify top performers and at-risk students
Allow users to choose custom visualizations
Generate automatic academic insights
Export results into a structured Excel report

 3. Key Features
 Subject-wise Analysis
Average marks per subject
Marks distribution using histograms
Outlier and consistency detection using box plots
Identification of hardest and easiest subjects

4. Student-wise Analysis
Total marks, average, and rank calculation
Grade assignment (A/B/C/D)
Student consistency score
Strongest and weakest subject per student
Identification of top and at-risk students

5.Interactive Visualization
Users can choose which plots to generate:
Histogram (marks distribution)
Bar chart (average per subject)
Box plot (outliers & spread)
All visualizations together

6.Automatic Insights
The system automatically generates insights such as:
Hardest and easiest subjects
Most consistent subject
Number of students at academic risk

7. Excel Report Generation
Creates a multi-sheet Excel file containing:
All student details
Subject-wise statistics
Top-performing students
At-risk students

8.Technologies Used
Python
Pandas – Data analysis
NumPy – Numerical computations
Matplotlib – Data visualization
OpenPyXL – Excel report generation
